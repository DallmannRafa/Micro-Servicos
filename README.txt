## Description

Starter repository for the Nest framework, a TypeScript framework for building efficient and scalable server-side applications.


## Installation

To install the dependencies, use the following command:

$ yarn install


## Running the application

To run the application, use one of the following commands:

$ yarn run start


## Testing

To run tests, use one of the following commands:

# Unit tests
$ yarn run test

# End-to-end tests
$ yarn run test:e2e

# Test coverage
$ yarn run test:cov
